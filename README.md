# Week 3 Example 2: Full Fighting Game

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.
To play via GitHub Pages, visit the link provided on the submission page.

**Player 1 Controls:**
Move the player left or right by pressing A / D, attack with F and block with G.

**Player 2 Controls:**
Move the player left or right by pressing left or right arrow key, attack with K and block with L.

Press **ENTER** to start or rematch.
